使用说明
==================
光盘相应文件夹下存放配书对应章节下范例的源程序及素材,文件夹的名称说明:
(1) ch开头的文件夹对应于书中各章的编号,如ch01文件夹存放了书中第1章所有的范例;
(2) 在ch开头的文件夹内有对应于每章编号从1开始的各个范例的文件夹,如1-1中就存放了第1章第一个范例;
(3) 在范例文件夹中(如1-1文件夹中)有三个文件夹,分别为"code",其中存放了该范例的源代码;"result",其中存放了该范例编译后生成的可执行文件，或者执行结果的截图; material",其中存放了范例中所用到的图片或音频资源.
